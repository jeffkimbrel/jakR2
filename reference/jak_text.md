# Text sizes and styles

Sets font sizes, families, and text angles for plot elements.

## Usage

``` r
jak_text(
  base_size = 10,
  plot_title_size = 12,
  axis_title_size = 11,
  base_family = ""
)
```

## Arguments

- base_size:

  Base font size. Default `10`.

- plot_title_size:

  Plot title font size. Default `12`.

- axis_title_size:

  Axis title font size. Default `11`.

- base_family:

  Base font family. Default `""` (system default).

## Value

A ggplot2 theme object
