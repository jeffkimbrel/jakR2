# Legend styling

Sets legend position and text sizes.

## Usage

``` r
jak_legend(legend_position = "bottom", legend_size = 10)
```

## Arguments

- legend_position:

  Legend position. One of `"bottom"`, `"top"`, `"left"`, `"right"`, or
  `"none"`. Default `"bottom"`.

- legend_size:

  Font size for legend text and title. Default `10`.

## Value

A ggplot2 theme object
