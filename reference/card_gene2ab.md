# Get gene information about an antibiotic

Get gene information about an antibiotic

## Usage

``` r
card_gene2ab(card, gene)
```

## Arguments

- card:

  output from
  [`card_parse()`](https://jeffkimbrel.github.io/jakR2/reference/card_parse.md)

- gene:

  character vector of gene name(s)
