# Filter an ft object by minimum count and sample

Filter an ft object by minimum count and sample

## Usage

``` r
## S7 method for class <jakR2::ft>
filter_asv_table(x, min_count = 2, min_sample = 2)
```

## Arguments

- x:

  An ft object

- min_count:

  Minimum count for an ASV to be retained (default is 2)

- min_sample:

  Minimum number of samples an ASV must be present in to be retained
  (default is 2)
