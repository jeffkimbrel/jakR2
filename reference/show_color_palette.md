# showColorPalette

showColorPalette

## Usage

``` r
show_color_palette(pal, labels = FALSE, label_angle = 0, alpha = 1)
```

## Arguments

- pal:

  A color palette

- labels:

  Boolean for whether to show the rgb value as a label

- label_angle:

  The rotational angle for the label

- alpha:

  A numeric value between 0 and 1 for the transparency of the colors
