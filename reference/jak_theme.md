# JAK ggplot theme

Modify theme of ggplots

## Usage

``` r
jak_theme(
  base_size = 10,
  plot_title_size = 12,
  axis_title_size = 11,
  base_color = "gray30",
  plot_title_color = "gray30",
  axis_title_color = "gray30",
  bg_color = "transparent",
  grid_color = "gray80",
  border_width = 0.5,
  strip_size = 10,
  strip_color = "transparent",
  strip_line = 0,
  legend_size = 10,
  legend_position = "bottom",
  base_family = ""
)
```

## Arguments

- base_size:

  Font size for most things

- plot_title_size:

  Font size for plot titles

- axis_title_size:

  Font size for axis titles

- base_color:

  Color for most text and lines

- plot_title_color:

  Color for plot titles

- axis_title_color:

  Color for axis titles

- bg_color:

  background color

- grid_color:

  Color for grid lines

- strip_size:

  Font size for facet strip titles

- strip_color:

  Color for facet strip titles

- strip_line:

  Line width for facet strip titles

- legend_size:

  Font size for legend

- legend_position:

  Position of legend

- base_family:

  Font family
