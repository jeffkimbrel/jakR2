# Get gene information about an antibiotic

Get gene information about an antibiotic

## Usage

``` r
card_ab2gene(card, ab, return = "df")
```

## Arguments

- card:

  output from
  [`card_parse()`](https://jeffkimbrel.github.io/jakR2/reference/card_parse.md)

- ab:

  character vector of antibiotic name(s)

- return:

  "df" to return a dataframe, "gene" to return a character vector of
  gene names
