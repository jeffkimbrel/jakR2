# Grid lines

Controls panel grid line appearance. Defaults to no grid lines.

## Usage

``` r
jak_grid(grid_color = "transparent", border_width = 0.5)
```

## Arguments

- grid_color:

  Color for major grid lines. Default `"transparent"` (no grid).

- border_width:

  Panel border line width. Default `0.5`.

## Value

A ggplot2 theme object
